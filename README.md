# Задачи для интервью

У тебя есть интерфейс Getter, который по адресу (address) и ключу (key) пытается получить значение. Но заранее неизвестно, на каком адресе лежит нужное значение, и сетевые вызовы могут падать

Нужно реализовать функцию Get, которая:

* Вызывает Getter.Get() параллельно для каждого адреса

* Возвращает первый успешный ответ

* Если все запросы завершились ошибкой, возвращает ошибку
